# Ragent_Agentic AI

This is a multi-agent AI project that performs tasks based on information extracted from web pages. It uses a web crawler to collect data, stores it as PDF files, and then uses a multi-agent system built with LangGraph to process and reason about the information.

## Features

- **Web Crawling**: Gathers information from specified websites and creates a local knowledge base.
- **Multi-Agent System**: Utilizes multiple AI agents that collaborate to perform complex tasks.
- **LangGraph Orchestration**: Manages the state and flow of information between agents using LangGraph.
- **RAG (Retrieval-Augmented Generation)**: Employs a RAG pipeline to retrieve relevant information from the collected documents to answer queries.

## Project Structure

```
/
├── data/                   # Stores PDF files generated by the crawler
├── data_collect/
│   └── crawling.py         # Script for web crawling and data collection
├── langgraph_multiagent.py # Main script for the multi-agent system
├── requirements.txt        # Project dependencies
└── README.md               # This file
```

## Installation

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd Ragent_Agentic_AI
    ```

2.  **Create and activate a virtual environment**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

3.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  **Collect Data**
    Run the crawling script to gather information from websites and populate the `data/` directory.
    ```bash
    python data_collect/crawling.py
    ```

2.  **Run the Multi-Agent System**
    Execute the main script to start the agentic workflow.
    ```bash
    python langgraph_multiagent.py
    ```

# Ragent_Agentic AI (한국어)

웹 페이지에서 추출한 정보를 기반으로 작업을 수행하는 멀티 에이전트 AI 프로젝트입니다. 웹 크롤러를 사용하여 데이터를 수집하고 PDF 파일로 저장한 다음, LangGraph로 구축된 멀티 에이전트 시스템을 사용하여 정보를 처리하고 추론합니다.

## 주요 기능

- **웹 크롤링**: 지정된 웹사이트에서 정보를 수집하여 로컬 지식 베이스를 생성합니다.
- **멀티 에이전트 시스템**: 여러 AI 에이전트가 협력하여 복잡한 작업을 수행합니다.
- **LangGraph 오케스트레이션**: LangGraph를 사용하여 에이전트 간의 상태와 정보 흐름을 관리합니다.
- **RAG (Retrieval-Augmented Generation)**: 수집된 문서에서 관련 정보를 검색하여 쿼리에 답변하는 RAG 파이프라인을 사용합니다.
